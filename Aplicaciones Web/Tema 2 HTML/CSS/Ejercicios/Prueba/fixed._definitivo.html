POSICIONAMIENTO FIJO

Similar al posicionamiento absoluto,

			pero cuando el usuario desplaza la página las 
			cajas se mantienen

			fijas en el mismo lugar.
			 Esto es útil para encabezados o pies de página

			que se mantienen inalterable a lo largo de un
			documento externo.

			También es posible aplicar el posicionamiento 
			fixed a un menú de navegación 

			lateral, que necesitemos esté siempre presente; 
			para los famosos widgets sociales,

			o un pequeño menú de ayuda al usuario.
